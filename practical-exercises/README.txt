# Practical Exercises - Object-Oriented Programming (OOP) in Python

This folder contains hands-on exercises designed to apply the principles of Object-Oriented Programming (OOP) in Python. All exercises are self-developed to reinforce key OOP concepts through practical problem-solving.

## Exercises:
1. Exercise 01: Creating a basic class (`Coche`)
2. Exercise 02: Implementing inheritance (`Vehicle`)
3. Exercise 03: Exploring polymorphism and method overriding
4. Exercise 04: Encapsulation and access control

Each script in this folder represents an attempt to apply OOP techniques in real-world scenarios. Feel free to explore and modify the exercises as you enhance your understanding of OOP.
